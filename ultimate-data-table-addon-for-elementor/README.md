A powerful Elementor addon that allows you to create fully customizable, responsive, and feature-rich data tables using DataTables.js.

## Description ##

**Ultimate Data Table Addon For Elementor** is the easiest way to build beautiful and functional data tables inside Elementor. Whether you need to display pricing, product info, or any structured data, this plugin gives you complete control over design, functionality, and responsiveness.

### Key Features: ###

🔢 **Entries Per Page**  
Let users control how many rows are displayed per page with a customizable “Show Entries” dropdown.  

🔍 **Live Search**  
Instant table filtering with a built-in search box. Users can find the information they need instantly.  

⏭ **Pagination**  
Full pagination support with multiple styles to match your design.  

↕ **Column Ordering**  
Sortable columns with ascending and descending order.  

📊 **Table Information**  
Show or hide table info, such as “Showing 1 to 10 of 120 entries.”  

🏷 **Custom Labels**  
Modify search label, no-data message, pagination text, and more to match your site’s language and style.  

🎨 **Uniform Cell Styling**  
Style every table cell consistently using Elementor controls.  

↕ **Rowspan Support**  
Merge table cells vertically to create grouped data structures.  

↔ **Colspan Support**  
Merge table cells horizontally for flexible layouts.  

📱 **Fully Responsive**  
Control how tables behave on all screen sizes, directly from Elementor.  

🧰 **Full Elementor Control Panel**  
All features are adjustable through Elementor — no coding required.  


**Data Source:**  
- Tables are powered by Elementor repeater fields, making it easy to manage dynamic data.

## Installation ##

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.  
2. Activate the plugin through the ‘Plugins’ screen in WordPress.  
3. Ensure Elementor is installed and activated.  
4. Add a new “Ultimate Data Table” widget in Elementor and start customizing your tables.

## Frequently Asked Questions ##
 
= 🔰 What is this plugin and what does it do? =  
* This is an addon for Elementor that lets you build fully functional, styled data tables with features like live search, pagination, sorting, and responsive behavior, all without coding.

= 🔰 Do I need to know code to use it? =  
* No. All controls and settings are handled through Elementor’s interface, so you don’t need any coding skills.

= 🔰 Is it responsive on mobile devices? =  
* Yes. You have control over how your tables behave and display across different screen sizes directly from Elementor.

= 🔰 Does this plugin work without Elementor? =  
* No. It’s an extension built specifically for the Elementor page builder, so Elementor must be installed and activated.

= 🔰 What table features are included? =  
It supports:
* entries-per-page selector
* live search filtering
* pagination with style options
* sortable columns
* custom table info (like “showing 1 to 10 of 50”)
* custom labels/messages
* rowspan and colspan for merged cells
* consistent cell styling via Elementor controls.

= 🔰 Can I customize how the table looks? =  
* Yes. Styling options for cell fonts, borders, colors, spacing, and more are available in the Elementor panel.

= 🔰 Where do table entries come from? =  
* Table content is powered by Elementor repeater fields, so you can manage structured dynamic data through those fields.

= 🔰 Are there any requirements? =  
Yes the plugin requires :
* WordPress 6.3 or newer
* PHP 7.4 or newer
* Elementor installed and activated.

= 🔰 Is there a limit on how many tables or rows I can create? =  
* No. You can create as many tables and rows as you need. (There’s no mention of caps in the official plugin doc.)

= 🔰 Where can I find help or support? =  
* Support is available through the plugin’s support forum on WordPress.org.
