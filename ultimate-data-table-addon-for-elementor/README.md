## Description ##

**Ultimate Data Table Addon For Elementor** is the easiest way to build beautiful, functional, and fully responsive data tables directly inside Elementor. Powered by DataTables.js, it gives you complete control over your table’s content, behavior, and design — all from the familiar Elementor panel, without writing a single line of code.

Whether you’re displaying pricing plans, product specs, schedules, comparison tables, or any structured data, this plugin has you covered. Add live search, pagination, column sorting, merged cells, custom labels, and pixel-perfect styling in minutes. To get started even faster, insert a **pre-made template** from the built-in library and customize it to match your brand.

### Why you’ll love it ###

* 100% built for Elementor — everything is a native Elementor control.
* No coding needed — configure and style visually.
* Fully responsive on every screen size.
* Lightweight and fast, using the proven DataTables.js engine.
* Ready-made 06 templates to kickstart your design.

### Key Features ###

🔍 **Live Search**
Instantly filter table rows with a built-in search box so visitors find what they need right away.

⏭ **Pagination**
Full pagination with 7 selectable styles — Simple, Simple Numbers, Numbers Only, Full, Full Numbers, and more.

🔢 **Entries Per Page**
Let visitors choose how many rows to display with a customizable “Show Entries” dropdown.

↕ **Column Ordering**
Sortable columns with ascending and descending order.

📊 **Table Information**
Show or hide summaries like “Showing 1 to 10 of 120 entries,” with token support (`_START_`, `_END_`, `_TOTAL_`, `_MAX_`).

🏷 **Custom Labels**
Edit the search label, no-data message, pagination label, and info text to match your site’s language.

↕ **Rowspan Support**
Merge cells vertically to build grouped data structures.

↔ **Colspan Support**
Merge cells horizontally for flexible layouts.

🎨 **Deep Styling Controls**
Style the table globally and per-section — header, body, rows/columns, search box, entries dropdown, pagination, and table info. Control typography, colors, borders, padding, alignment, striped rows, and more.

📱 **Fully Responsive**
Control how tables behave and display across all screen sizes, directly from Elementor.

🧩 **Pre-Made Templates**
Insert ready-made data-table designs from the template library inside the Elementor editor and customize them instantly.

🧰 **Full Elementor Control Panel**
Every option lives inside Elementor — no coding required.

**Data Source:** Tables are powered by Elementor repeater fields, making dynamic data easy to manage.

## Installation ##

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install it directly from the WordPress **Plugins → Add New** screen.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Make sure **Elementor** is installed and activated.
4. Edit any page with Elementor, search for the **Data Table** widget, drag it in, and start building — or insert a pre-made template from the library.

## Frequently Asked Questions ##

= What does this plugin do? =
It’s an Elementor addon that lets you build fully functional, styled data tables with live search, pagination, sorting, merged cells, and responsive behavior — all without coding.

= Do I need to know how to code? =
No. Every setting is handled through Elementor’s interface, so no coding skills are required.

= Does it work without Elementor? =
No. It’s an extension for the Elementor page builder, so Elementor must be installed and activated.

= Is it responsive on mobile devices? =
Yes. You control how tables display and behave across screen sizes directly from Elementor.

= What table features are included in the free version? =
Live search, entries-per-page selector, pagination with multiple styles, sortable columns, custom table info, editable labels, rowspan and colspan for merged cells, full styling controls, and a pre-made template library.

= Are there pre-made templates? =
Yes. The plugin includes a template library inside the Elementor editor. The free version comes with 6 ready-made templates. Upgrading to Pro unlocks 27 additional premium templates.

= Can I put images, icons, or buttons inside a cell? =
That’s a Pro feature. The free version supports rich text cells. Ultimate Data Table Pro adds image, icon, and linked-button content inside any cell, each with its own styling options.

= Can I customize how the table looks? =
Yes. Fonts, colors, borders, spacing, alignment, striped rows, and more are all adjustable from the Elementor panel.

= Where does table content come from? =
Table content is powered by Elementor repeater fields, so you manage structured data through those fields.

= Is there a limit on how many tables or rows I can create? =
No. You can create as many tables and rows as you need.

= Where can I get support? =
Support is available through the plugin’s support forum on WordPress.org.
